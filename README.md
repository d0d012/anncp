# anncp — Neural Network in C++

A feedforward neural network implemented from scratch in C++, with a Python script for exporting pre-trained weights from PyTorch.

## Overview

The project builds a simple multi-layer perceptron in C++ using only the standard library. Weights are generated by a PyTorch model in Python and saved to a text file, which the C++ inference engine then loads and runs.

**Default architecture:** `2 → 4 → 1` (2 inputs, 4 hidden neurons with ReLU, 1 output)

## Project Structure

```
.
├── Matrix.h / Matrix.cpp       # 2D matrix with +, *, and ReLU operations
├── Layer.h / Layer.cpp         # Single fully-connected layer (weights + biases)
├── NeuralNetwork.h / .cpp      # Network composed of layers; forward pass + weight loading
├── main.cpp                    # Entry point: loads weights, runs a sample prediction
├── script.py                   # Exports random PyTorch weights to weights.txt
└── weights.txt                 # Weight file read by the C++ program
```

## Requirements

- C++17 compiler (g++ / clang++)
- Python 3 + PyTorch (only needed to regenerate weights)

## Usage

### 1. Generate weights

```bash
python3 script.py
```

This creates `weights.txt` with randomly initialised weights from a PyTorch model matching the C++ architecture.

### 2. Build the C++ program

```bash
g++ -std=c++17 -o ai_engine main.cpp Matrix.cpp Layer.cpp NeuralNetwork.cpp
```

### 3. Run inference

```bash
./ai_engine
```

The program loads `weights.txt`, feeds the input `[1.0, 0.5]` through the network, and prints the prediction.

## How It Works

1. `script.py` defines a `SimpleNet` (Linear → ReLU → Linear) in PyTorch, flattens all parameters layer by layer, and writes them to `weights.txt`.
2. `NeuralNetwork::loadWeightsFromFile` reads that file, filling weight and bias matrices for each layer in the same order.
3. `NeuralNetwork::predict` performs a forward pass: each hidden layer applies a linear transform followed by ReLU; the output layer applies only the linear transform.
